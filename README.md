# Node Server Starter

THIS IS A WORK IN PROGRESS, DON'T USE THIS FOR ANYTHING

A small repo to get started with Node, Express, Sequelize, and Postgres.

## Running Locally:
- install node & postgres
- clone repo
- `npm install`
